# Версия

Текущая версия: **0.2.2**

Этап: **Поиск в Web Worker с паузой и продолжением**

Состояние:

- основа приложения: готова;
- версия и схема базы Bitcoin-профилей: готовы;
- стандартные BIP44/BIP49/BIP84/BIP86: готовы;
- проверка английской BIP39 seed-фразы и checksum: готова;
- необязательная BIP39 passphrase: поддерживается;
- локальная генерация Bitcoin mainnet-адресов: готова;
- поиск нескольких известных адресов за один запуск: готов;
- автоматический подбор совместимых профилей по типам адресов: готов;
- удаление дублей и отдельные карточки совпадений: готовы;
- вычисления вынесены из интерфейса в Web Worker;
- пауза, продолжение, остановка и очистка формы: готовы;
- экспорт очищенного отчёта: ещё не реализован;
- сетевой поиск активности: не реализован и отключён по умолчанию.

Во время поиска seed-фраза передаётся только выделенному рабочему потоку текущей страницы. При завершении или очистке worker уничтожается. Приложение не использует localStorage, IndexedDB, аналитику или внешние API.
